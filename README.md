# kaosfield

My site [kaosfield](https://www.kaosfield.net) contents and builder for distribution.

Build with:

- [pug](https://pugjs.org/)

## How to build

Install Node.js and dependencies.

```sh
asdf install
npm i
```

Build.

```sh
npm run build
```

Deployment.

```sh
cp deploy.sh.example deploy.sh
# Edit deploy.sh
npm run deploy
```
