お知らせ　2008-06-21(Sat)
移行先で書かれていた日記が更に移行しました．SereneBach というブログツールを使って書くようになったのです．現在日記を書いてすらいません．日記ログがあるだけで，情報発信は Twitter や GitHub や Qiita です．というわけで，まったく更新をしないこのページですがリンクの差し替えだけは一応やっておきます．
こちら (別窓で開きたい方はこちら) からどうぞ．

もうまったく存在価値のないページですし，ここの消滅もそろそろ考えた方がいいかも知れませんね．

やり込みコンテンツ等のコンテンツそのものの存在価値もないっちゃないですが，あれはまぁなんというか「小学生の頃作った夏休みの自由工作」みたいな感じで思い入れがあるので，さすがに消したくはありません(^^;　じゃあ web から削除してローカルに保存しておけばいいじゃん，と仰る方もいるかもしれませんが，ここで私は次のように考えるのです．

夏休みの自由工作は，作ったことに意義があったのだ，と．つまり，役に立つ立たないは二の次で，とにかく作ったという事実が大事なのだと．

そういう考え方をしたときに，私にとってやり込み攻略レポートというのは「書き上げてから web で公開することに意義があった」となるのです．だからその点を，web で公開する段階まで含めた状態を「保存」しておきたい，と．

なので，未来永劫自分にとってしか意味の無いコンテンツではあると思いますが，web から削除するということは決してしないと思います．何らかの形で保存はすると思います．

どこにどういう形で保存するかはこれからゆっくり考えていきます．そして，その考えが煮詰まったときこそが，web サイト「桃源郷」が消滅するときなのです．
